# User guide

This page refers to the latest 
[nv_typewriter](https://github.com/peter88213/nv_typewriter/) release.

The plugin adds an **Edit in distraction free mode** entry to the *novelibre* **Tools** menu. 

---

# Operation

- Click **Tools > Edit in distraction free mode** to enter the distraction free editing mode.
- Close the editor window to finish the editing session.
